# Offline APK Demo First

## Lesson

For this project, the most valuable integration milestone is not a polished UI.
It is a repeatable offline APK demo: open app, choose local image, generate
histogram, show time.

## Use When

- Android shell work starts.
- H5 and Android need integration.
- Final acceptance or defense material is being prepared.

## What To Do

- Bundle local H5 assets.
- Verify first screen without network.
- Prepare local test images.
- Keep the demo path short and deterministic.
