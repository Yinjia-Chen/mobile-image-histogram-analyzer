# Backend Scope Creep

## Failure Mode

Adding login, upload service, database, cloud storage, or remote image hosting
can make the project look bigger while weakening the actual course deliverable.

## Why It Hurts

The task scores a mobile histogram program, result accuracy, speed display,
project documents, and defense. Backend scope adds risk without helping the core
acceptance path.

## Avoid By

- Keeping image processing local.
- Treating backend ideas as future extensions.
- Updating docs when scope is intentionally changed by the user.
